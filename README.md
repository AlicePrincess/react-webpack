# tuso

NGS项目

./components/    tuso项目公共组件
./utils/         tuso项目公共方法

> share —— 图说分享
> web —— 图说网页版
> company —— 官网
> children —— 六一儿童节
> admin —— 图说管理系统
> mobile —— 图说移动版

## share
图说分享

### mac
> 运行：sudo npm run start-share
> 打包：sudo npm run build-share

### win
> 运行：npm run start-win-share
> 打包：npm run build-win-share
